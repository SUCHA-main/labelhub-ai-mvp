# LabelHub AI MVP 演示脚本

## 一句话介绍

LabelHub AI MVP 是一个面向数据标注团队的 AI 辅助标注平台原型，用动态表单、AI 预审和人工审核完成从任务创建到质量闭环的演示流程。

## 演示账号

登录页无需密码，用户名可填写 `demo`，切换以下角色进入不同工作台。

| 角色 | 演示用途 |
| --- | --- |
| admin | 创建任务、查看 dashboard 和全部任务 |
| annotator | 查看待标注任务，提交动态表单标注结果 |
| reviewer | 查看待审核任务，参考 AI 预审结果进行通过或驳回 |

## 演示流程

1. 以 `admin` 登录，展示 dashboard 中的任务总数、待处理任务数、通过率和 AI 风险提示数。
2. 在管理员工作台查看 3 条预置任务：情绪分类、商品标题质量判断、内容安全审核。
3. 创建一条新任务，强调 schema 可以动态决定标注表单字段。
4. 退出后以 `annotator` 登录，选择一条待标注任务，填写表单并提交。
5. 提交后说明后端会调用 `mockAIReview`，生成风险等级、置信度、问题提示和修改建议。
6. 退出后以 `reviewer` 登录，查看待审核任务和 AI 预审卡片。
7. 审核员填写审核意见，选择通过或驳回，dashboard 状态随之更新。

## 项目亮点

- 动态 schema 驱动标注表单，适配不同类型文本标注任务。
- AI 预审结果结构化展示，让评委能快速看到“模型辅助质检”的价值。
- 三角色流程完整，覆盖管理员、标注员、审核员的协作闭环。
- JSON 文件存储和 Docker 部署降低演示成本，适合比赛现场快速运行。
- Dashboard 增加通过率、AI 风险提示数和待处理任务数，更贴近运营看板。
